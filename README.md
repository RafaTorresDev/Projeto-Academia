# Sistema de Monitoramento de Alunos - Academia

## 📌 Descrição
Sistema desenvolvido em Java para gerenciamento de alunos em uma academia.

## 🛠 Tecnologias Utilizadas
- Java
- Programação Orientada a Objetos

## 🧠 Conceitos Aplicados
- Encapsulamento
- Herança
- Polimorfismo
- Organização em classes

## 🚀 Funcionalidades
- Cadastro de alunos
- Controle de planos
- Gerenciamento de membros

## ▶ Como Executar
Executar a classe Main.java
